## Screenshot Specifications

### Recommended Screenshot Settings:
- **Resolution:** 1920×1080 (standard laptop)
- **Format:** PNG or JPG
- **Annotations:** Use arrows, circles, numbers to highlight key elements
- **Color overlays:** Use for emphasis (e.g., red box around important number)
- **Contrast:** Ensure text is readable

### Annotation Best Practices:
1. Add numbers to reference in caption
2. Use arrows for "before" → "after"
3. Circle important data points
4. Add red box for critical information
5. Use 16-18pt font for annotations

---
